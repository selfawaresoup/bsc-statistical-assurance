# BSc thesis by Esther Weidauer

Psychology BSc thesis regarding statistical asssurance for estimating sample sizes and power

## Rendering documents

### Incremental render

```
make all
```

### Clean render

```
make clean && make all
```

### Rendering individual documents

```
make expose
```

```
make thesis
```

### Requirements

- TeXLive (with `latexmk`)
- GNU Make (or equivalent)
